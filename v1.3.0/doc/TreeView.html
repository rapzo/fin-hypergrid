Class: TreeView

TreeView

This is a simple helper class to set up the tree-view data source in the context of a hypergrid.

It includes methods to:

  • Build a new pipeline with DataSourceTreeview and appropriate sorter and filter.
  • Perform the self-join and rebuild the index to turn the tree-view on or off, optionally hiding the ID columns.

Methods

deleteRow(ID, adoptiveParentIDopt, keepParentopt, keepDrillDownopt) → {number}

Delete a row and it's children.

Requires that the row-by-id API is installed.

Alternatively, you can reassign the children to another row (see adoptiveParentID below).

After you're done with all your row manipulations, you must call:

grid.behavior.reindex();
grid.behaviorShapeChanged();
grid.repaint(); // call this eventually
Parameters:
Name Type Attributes Description
ID number

ID of the row to delete.
adoptiveParentID null | number <optional>

ID of the row to reassign the orphaned children to. If null, reassigns to top-level. If omitted, the orphans are recursively deleted. If omitted, the remaining parameters are promoted one position.
keepParent boolean <optional>

Just delete (or reassign) children but keep the parent.
keepDrillDown boolean <optional>

Keep drill down control on the kept parent.
Returns:

Total rows deleted.

Type
number

setRelation(join) → {DataSourceTreeView}

Build/unbuild the tree view.

"Joins" the table to itself through the ID and parent ID columns using the options given to the constructor (see above).

Reconfigures the data model's data pipeline for tree view join; restores it when unjoined.

Also saves and restores some grid properties:

  • Tree column is made non-editable.
  • Tree column is made non-selectable so clicking drill-down controls doesn't select the cell.
  • Row are made selectable by clicking in row handles only so clicking drill-down controls doesn't select the row.
Parameters:
Name Type Description
join boolean

If truthy, turn tree-view ON. If falsy (or omitted), turn it OFF.
Returns:

Indicates joined state; undefined indicates unjoined.
